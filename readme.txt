Forex Intraday Signal Bot with Telegram Alerts

HOW TO USE:
1. Export 15-minute data from MT4 for:
   - EURUSD as EURUSD_M15.csv
   - XAUUSD as XAUUSD_M15.csv

2. Put the CSV files in the same folder as main.py

3. Run:
   > python main.py

The bot will:
- Analyze using Moving Average, Bollinger Bands, and RSI
- Detect intraday signals (BUY/SELL)
- Send alerts directly to your Telegram using your BOT
